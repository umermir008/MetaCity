# NeuroAI - Smart AI Automation Platform

🚀 **A fully responsive, 3D animated single-page SaaS website for an AI startup**

## ✨ Features

### 🎨 Design & Animation
- **Futuristic Dark Theme** with neon blue/purple glow effects
- **3D Animated Elements** with GSAP-powered smooth transitions
- **Parallax Scrolling** backgrounds for immersive experience
- **Floating AI Brain** animation in hero section
- **Interactive 3D Cards** with hover effects and neon borders
- **Smooth Scroll Triggers** for all section animations

### 📱 Responsive Design
- **Mobile-First** approach with full responsiveness
- **Tablet & Desktop** optimized layouts
- **Touch-Friendly** interactions for mobile devices
- **Landscape Mode** optimizations
- **High DPI Display** support

### 🎯 Interactive Elements
- **Working Contact Form** with validation and animations
- **Ripple Button Effects** on click interactions
- **Auto-Scrolling Testimonials** carousel
- **Video Showcase** with YouTube embed
- **Mobile Navigation** menu with smooth animations
- **Custom Cursor** effects (desktop)

### ⚡ Performance
- **Fast Loading** with optimized animations
- **Lazy Loading** for images
- **Throttled Scroll Events** for smooth performance
- **Accessibility Support** with reduced motion preferences
- **SEO Optimized** structure

## 🔧 Technical Stack

- **HTML5** - Semantic structure
- **Tailwind CSS** - Utility-first styling
- **Vanilla JavaScript** - Performance-focused interactions
- **GSAP** - Professional animations
- **Font Awesome** - Icon library
- **Google Fonts** - Inter typography

## 📁 Project Structure

```
NeuroAI/
├── index.html          # Main website file
├── script.js           # JavaScript animations & interactions
├── responsive.css      # Additional responsive styles
└── README.md          # Project documentation
```

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Start a local server**:
   ```bash
   python -m http.server 8000
   ```
3. **Open your browser** to `http://localhost:8000`
